## 4. Frequently Asked Questions (FAQ)

**Q: How does Snapshot pricing work?**  
A: Pricing is based on the actual size of the data you are backing up, not the total capacity of the source disk. Incremental snapshots only charge for the new or changed data added since the last capture.

**Q: Can I use my server while a snapshot is being created?**  
A: Yes. The process is highly optimized and runs in the background. You can continue normal operations with no impact on performance or availability.

**Q: What is the difference between a Full and Incremental snapshot?**  
A: The first snapshot of any disk is a "Full" snapshot (containing all written data). Every snapshot after that is "Incremental," meaning it only stores the data blocks that have changed since the previous snapshot.

**Q: Are snapshots encrypted?**  
A: Yes. If your source disk is encrypted, the resulting snapshot will be encrypted using the same standards, ensuring cumulative security for your backups.
